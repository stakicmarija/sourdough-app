# Sourdough Journal 

AI-powered web app for analyzing sourdough bread using Claude.

## Features

*  Upload a photo of your sourdough bread
*  Get AI-generated feedback on:

  * crumb structure
  * crust quality
  * fermentation
  * oven spring
*  Continue the conversation with follow-up questions (multi-turn chat)
*  Streaming responses for real-time feedback

## Tech Stack

* Python
* Streamlit
* Anthropic Claude API

## How it works

The app sends the uploaded image and notes to Claude, which analyzes the bread and returns structured feedback.
The conversation is maintained through session state, enabling follow-up questions with full context.
